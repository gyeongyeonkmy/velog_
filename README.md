# velog_
🌱 벨로그 글도 쓰고 잔디도 심고

https://velog.io/@gyeongyeonk/posts의 글이 posts 폴더에 아카이빙 됩니다

Velog posts are archived under `posts/<year>/` and synced automatically from RSS.


## Local setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

If VS Code still shows `Import "feedparser" could not be resolved`, select the `.venv` interpreter for this workspace.
